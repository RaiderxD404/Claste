# Claste
Un modpack para la version 1.16.5.


# Como lo descargo e instalo?

Pues facil, con el modpack descargado, lo extraes y te aparecera una carpeta, esta carpeta es lo que debe haber en tu .minecraft

# Lo recomendado

Pues, no lo pongas en tu .minecraft, te ahorras tener que borrar el .minecraft y mas problemas, puedes hacer una carpeta y decirle a tu launcher que los archivos de configuracion se generen ahi.


Ejemplo: https://youtu.be/U1fMPFCY4bA

En vez de prender el juego cuando se haya hecho la carpeta y puesto la version con tu launcher, debes borrar los archivos que esten dentro de la carpeta y colocar todos los archivos del modpack.
